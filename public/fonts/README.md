# Brand Font Setup

Place your **Sansserifflf-Demibold** font files here:

- `Sansserifflf-Demibold.woff2`  (preferred, smaller)
- `Sansserifflf-Demibold.woff`   (fallback)

These are referenced in `app/layout.tsx` via `next/font/local`.

Until placed, the site falls back to Arial/Helvetica Neue gracefully.
